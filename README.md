## 🚀 Tech Stack
*   **Frontend:** HTML5, CSS3 (Premium Minimalist), Vanilla JavaScript (ES6+)
*   **Backend:** Google Apps Script (GAS) API
*   **Database:** Google Sheets
*   **Libraries:** Chart.js, html2pdf.js, FontAwesome

## 📦 Key Features
### 📊 Dashboard & Analytics
- **Sales Visualization:** กราฟยอดขายและสัดส่วนสาขาแบบ Real-time
- **Smart Stock Analysis:** กราฟแท่งซ้อนแจ้งเตือนสต็อกสินค้าใกล้หมด

### 🛠️ Operation Tools
- **Order Management:** ระบบจัดการสถานะออเดอร์ (Packed / Success) พร้อม God Mode
- **Menu Drag & Drop:** จัดการเมนูและหมวดหมู่ด้วยการลากวาง (Nitro Engine)
- **Deep Search:** ระบบค้นหาออเดอร์แบบเจาะลึก

### 🖨️ Reporting
- **Thermal Receipt:** ระบบพิมพ์ใบเสร็จขนาดเล็ก
- **A4 Summary Report:** รายงานสรุปยอดขายแนวตั้ง (PDF)

## 🔧 Installation & Setup
1.  Clone repository นี้
2.  เปิดไฟล์ `manager.html` ใน Browser เพื่อเริ่มใช้งาน
3.  เชื่อมต่อกับ Google Apps Script Endpoint ที่ระบุใน `script.js`
